  1. Write a program to find the factorial of any given number using Javascript.

    Answer:

    The factorial of a number is the product of all positive integers less than or equal to that number.

    JavaScript Code:

    
function factorial(n) {
  let result = 1;
  for (let i = 1; i <= n; i++) {
    result *= i;
  }
  return result;
}

// Example usage
let number = 5;
console.log("The factorial of " + number + " is: " + factorial(number));

    Output: The factorial of 5 is: 120

    Explanation: The loop multiplies numbers from 1 to the given number (5), so 1 × 2 × 3 × 4 × 5 = 120.

  38. Write a PHP program to print the largest and smallest numbers among 10 elements of an array.

    39. Answer:

            <?php
$numbers = array(12, 45, 67, 2, 89, 34, 22, 10, 99, 5);
$max = max($numbers);
$min = min($numbers);

echo "Largest number: " . $max . "<br>";
echo "Smallest number: " . $min;
?>

    41. Answer:

  42. What is event handling in JavaScript?

    43. Answer:

    Event handling in JavaScript is the process of executing code when specific events occur in a webpage, such as clicks, keypresses, or page loads. This is done using event listeners or event attributes like onclick.

    Example:

    <button onclick="alert('Button clicked!')">Click Me</button>
